# README

The Monkey programming language.

Implemented as part of the book "Writing An Interpreter In Go" by Thorsten Ball.

![Monkey Demo](demo.gif)

## Data types

- Integer
- Boolean
- Null
- String
- Array
- Hash

## Features

- Let statements
- If/else statements
- Return statements
- Function literals
- Function calls
- Higher-order functions
- Closures
- String concatenation
- Array literals
- Array indexing
- Hash literals
- Hash indexing
- Built-in functions
  - `len`
  - `first`
  - `last`
  - `rest`
  - `push`
  - `print`
